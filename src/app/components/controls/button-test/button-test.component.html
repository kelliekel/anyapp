Button

Extends formcontrol (description about formcontrols)

Example

This is a {{buttonStyle}} {{buttonColor}} button

Code

<aa-comp-button [tooltip]="'This is the optional tooltip'" [color]="'{{buttonColor}}'" [style]="'{{buttonStyle}}'">
  This is a button
</aa-comp-button>

Options

Property Type Extra
style string (default: 'flat') {{s}} |
color string (default: 'primary') {{s}} |
type string (default: 'button') Buttontype; for example: button or submit
timeout boolean (default: true) Indicates whether or not the button should be disabled after a click for a certain amount of time
